CodeBlocks配置简单python环境

CodeBlocks配置Python可以用于对Python的学习，方便快捷
具体步骤如下

安装python

下载python

配置CodeBlocks

下载CodeBlocks
打开CodeBlocks选中菜单栏的Tools–Configure tools

点击Add

添加工具

  • Name:填写工具名，可任意填写
  • Executable:选择安装的python路径下的python.exe
  • Parameters:填写窗口底部的第四行命令（如图）
  • Launching options选择第一项
    设置完成后点击OK确定

    再选中菜单栏的Setting–Editor

    为工具添加快捷键

  • 在左栏选中Keyboard shortcuts
  • 在Commands栏中选中Tools里你命名的工具( 图为R-py)
  • 在右边New shortcut栏中填入你想设置的快捷键(推荐F5)并点击Add添加
    设置完成后点击OK确定

测试是否成功配置

新建.py后缀文件，编写代码后保存，按下设定的快捷键测试是否成功
